### Health
- Starts at `100`
- Damaged by mob fights and Creeper explosions
- Does **not** regenerate automatically
- Reaching `0` triggers the death screen

### Hunger

- Drained by: Mining (`-10`), Fighting (`-15`), Creeper (none)
